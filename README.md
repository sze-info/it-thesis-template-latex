# GIVK Szakdolgozat/Diplomamunka LaTeX sablon
Folyamatosan karbantartott LaTeX szakdolgozat illetve diplomamunka sablon az SZE-GIVK hallgatói számára. A [BME-VIK LaTeX sablonján](https://github.com/FTSRG/thesis-template-latex) alapul, amelyet karunk elvárásainak megfelelően alakítottam át. A projekt Overleaf-en is megtalálható: https://v2.overleaf.com/read/jwykzctmcfsh (csak olvasható link, ami nem feltétlenül aktuális, majd lesz CI-vel készülő pdf).

# Hogyan használd?
A sablon legújabb verziója mindig egy egyetemi GitLab repo-ban érhető el. Ezt .zip fájlként letöltve, vagy akár Git segítségével forkolva kezdheted a munkád. A sablon bőségesen el van látva útmutató kommentekkel.

# Segíts a fejlesztésben!
Amennyiben hibát találsz a sablonban, jelezd mint `Issue`, majd miután megbeszéltük a problémát, kijavíthatod és beadhatsz egy `Merge request`-et.
